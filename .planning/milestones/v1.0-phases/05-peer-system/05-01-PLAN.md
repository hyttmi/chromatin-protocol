3. **Create SyncProtocol** (`src/sync/sync_protocol.h`, `src/sync/sync_protocol.cpp`):
   The SyncProtocol class handles sync logic between two nodes. It operates at a higher level than Connection, using BlobEngine for queries and Connection for transport.

   Key design decisions:
   - SyncProtocol is NOT a coroutine class. It provides synchronous helper methods for the sync algorithm (hash collection, diffing, expiry filtering). The coroutine orchestration lives in PeerManager (Plan 02).
   - This keeps SyncProtocol fully testable without Asio.

   Public API:
   ```cpp
   namespace chromatin::sync {

   struct SyncStats {
       uint32_t blobs_sent = 0;
       uint32_t blobs_received = 0;
       uint32_t namespaces_synced = 0;
   };

   class SyncProtocol {
   public:
       explicit SyncProtocol(engine::BlobEngine& engine, storage::Clock clock = storage::system_clock_seconds);

       // Check if a blob is expired given current time
       static bool is_blob_expired(const wire::BlobData& blob, uint64_t now);

       // Collect non-expired blob hashes for a namespace
       std::vector<std::array<uint8_t, 32>> collect_namespace_hashes(
           std::span<const uint8_t, 32> namespace_id);

       // Diff: return hashes in `theirs` not present in `ours`
       static std::vector<std::array<uint8_t, 32>> diff_hashes(
           const std::vector<std::array<uint8_t, 32>>& ours,
           const std::vector<std::array<uint8_t, 32>>& theirs);

       // Get blobs by hash list (for transfer)
       std::vector<wire::BlobData> get_blobs_by_hashes(
           std::span<const uint8_t, 32> namespace_id,
           const std::vector<std::array<uint8_t, 32>>& hashes);

       // Ingest received blobs (validates + stores, skips expired)
       SyncStats ingest_blobs(const std::vector<wire::BlobData>& blobs);

       // Encode/decode sync messages using FlatBuffers transport codec
       // NamespaceList payload: [count:u32][ns1:32B][seq1:u64]...[nsN:32B][seqN:u64]
       static std::vector<uint8_t> encode_namespace_list(
           const std::vector<storage::NamespaceInfo>& namespaces);
       static std::vector<storage::NamespaceInfo> decode_namespace_list(
           std::span<const uint8_t> payload);

       // HashList payload: [ns:32B][count:u32][hash1:32B]...[hashN:32B]
       static std::vector<uint8_t> encode_hash_list(
           std::span<const uint8_t, 32> namespace_id,
           const std::vector<std::array<uint8_t, 32>>& hashes);
       static std::pair<std::array<uint8_t, 32>, std::vector<std::array<uint8_t, 32>>>
           decode_hash_list(std::span<const uint8_t> payload);

       // BlobTransfer payload: [count:u32][len1:u32][blob1_flatbuf]...[lenN:u32][blobN_flatbuf]
       static std::vector<uint8_t> encode_blob_transfer(
           const std::vector<wire::BlobData>& blobs);
       static std::vector<wire::BlobData> decode_blob_transfer(
           std::span<const uint8_t> payload);
   };
   }
   ```

   Implementation notes:
   - `is_blob_expired`: returns `(blob.ttl != 0 && (blob.timestamp + blob.ttl) <= now)`
   - `collect_namespace_hashes`: calls `engine_.get_blobs_since(ns, 0)`, filters expired via `is_blob_expired`, computes blob_hash for each non-expired blob using `wire::blob_hash(wire::encode_blob(blob))`
   - `diff_hashes`: builds unordered_set from `ours` (using string_view over raw bytes), iterates `theirs`, collects those not in set
   - `get_blobs_by_hashes`: for each hash, calls `engine_.get_blob(ns, hash)`, collects hits
   - `ingest_blobs`: for each blob, checks `is_blob_expired` first, then calls `engine_.ingest(blob)`. Counts accepted blobs.
   - Message encoding uses simple binary format (NOT FlatBuffers tables for inner payloads -- they're already wrapped in TransportMessage). Use network byte order (big-endian) for u32/u64 fields. This matches the existing framing pattern.
   - Injectable clock for expiry tests (same pattern as Storage).
