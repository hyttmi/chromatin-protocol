   ```
   SIZE_IDX=$((ITER % 5))
   SIZES=(128 1024 10240 102400 1048576)
   SIZE=${SIZES[$SIZE_IDX]}
   # 50 blobs at rate 10 = 5 seconds per batch
   run_loadgen_bg 172.50.0.2 --count 50 --rate 10 --size $SIZE --ttl 3600
   ```

**Key implementation detail for RSS parsing:** `docker stats --no-stream --format '{{.MemUsage}}'` returns e.g. "45.2MiB / 7.77GiB". Extract first part, convert:
```bash
parse_rss_bytes() {
    local raw="$1"
    local num=$(echo "$raw" | grep -oP '[\d.]+' | head -1)
    local unit=$(echo "$raw" | grep -oP '[KMG]iB' | head -1)
    case "$unit" in
        KiB) echo "$(echo "$num * 1024" | bc | cut -d. -f1)" ;;
        MiB) echo "$(echo "$num * 1048576" | bc | cut -d. -f1)" ;;
        GiB) echo "$(echo "$num * 1073741824" | bc | cut -d. -f1)" ;;
        *) echo "0" ;;
    esac
}
```

**Part B: Update run-integration.sh**

Read the existing `run-integration.sh` and modify the test discovery to exclude `test_stress01_long_running.sh` from default runs. Add an exclusion in the filter application loop:

After the `mapfile -t ALL_TESTS` line that discovers test_*.sh files, add:
```bash
# Exclude long-running soak test from default runs (invoke explicitly)
EXCLUDED_TESTS=("test_stress01_long_running.sh")
```

In the filter loop where TESTS array is built, add an exclusion check:
```bash
TESTS=()
for t in "${ALL_TESTS[@]}"; do
    name="$(basename "$t")"
    # Skip excluded tests unless explicitly filtered
    if [[ -z "$FILTER" ]]; then
        local skip=false
        for excl in "${EXCLUDED_TESTS[@]}"; do
            if [[ "$name" == "$excl" ]]; then
                skip=true
                break
            fi
        done
        if [[ "$skip" == true ]]; then
            log "Skipping $name (long-running, use --filter stress01 to run explicitly)"
            continue
        fi
    fi
    if [[ -z "$FILTER" || "$name" == *"$FILTER"* ]]; then
        TESTS+=("$t")
    fi
done
```

This means: with no --filter, stress01 is skipped. With `--filter stress01`, it runs. With `--filter stress`, all stress tests including stress01 run.

Note: `local` is not valid outside a function in bash. Use a regular variable instead of `local skip`.
