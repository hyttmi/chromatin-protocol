1. In the top-level CMakeLists.txt, replace lines 28-32 (the ENABLE_ASAN option block) with a SANITIZER enum option:

```cmake
set(SANITIZER "" CACHE STRING "Sanitizer to enable: asan, tsan, or ubsan (mutually exclusive)")
set_property(CACHE SANITIZER PROPERTY STRINGS "" "asan" "tsan" "ubsan")

if(SANITIZER STREQUAL "asan")
    add_compile_options(-fsanitize=address -fno-omit-frame-pointer -fno-optimize-sibling-calls)
    add_link_options(-fsanitize=address)
    message(STATUS "AddressSanitizer enabled")
elseif(SANITIZER STREQUAL "tsan")
    add_compile_options(-fsanitize=thread -fno-omit-frame-pointer)
    add_link_options(-fsanitize=thread)
    message(STATUS "ThreadSanitizer enabled")
elseif(SANITIZER STREQUAL "ubsan")
    add_compile_options(-fsanitize=undefined -fno-omit-frame-pointer -fno-sanitize-recover=all)
    add_link_options(-fsanitize=undefined)
    message(STATUS "UndefinedBehaviorSanitizer enabled")
elseif(NOT SANITIZER STREQUAL "")
    message(FATAL_ERROR "Unknown SANITIZER value: ${SANITIZER}. Use asan, tsan, or ubsan.")
endif()
```

Key flags:
- `-fno-omit-frame-pointer`: better stack traces for all sanitizers
- `-fno-optimize-sibling-calls`: ASAN-specific, prevents tail-call optimization from hiding stack frames
- `-fno-sanitize-recover=all`: UBSAN-specific, abort on first UB finding (don't continue)
