<objective>
Add configurable namespace-scoped sync filtering. Operators specify which namespaces the node replicates via sync_namespaces config. The filter applies at sync Phase A (namespace list assembly) and at Data/Delete ingest. Empty list means replicate all (backward compatible). SIGHUP-reloadable.

Purpose: Enable operators to control which namespaces their node handles, supporting topology-aware deployment where different nodes serve different namespace subsets.
Output: Namespace filter at sync Phase A and ingest, configurable via JSON, SIGHUP-reloadable.
</objective>

    ```cpp
    std::array<uint8_t, 32> hex_to_namespace(const std::string& hex) {
        std::array<uint8_t, 32> result{};
        for (size_t i = 0; i < 32; ++i) {
            auto byte_str = hex.substr(i * 2, 2);
            result[i] = static_cast<uint8_t>(std::stoul(byte_str, nullptr, 16));
        }
        return result;
    }
    ```

    ```cpp
    sync_namespaces_.clear();
    ```
