## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-122-01 | Spoofing | blob.fbs schema | mitigate | Removing inline pubkey prevents a new class of attack where forged "pubkey that matches schema-valid shape" bypasses verify; resolution now MUST go via owner_pubkeys DBI (Plan 02/04). |

| T-122-07 | Integrity | Schema-level invariant | mitigate | The schema enforces that EVERY blob carries a `signer_hint` (no optional path to pre-122 shape); enforced at the FlatBuffers decode layer (Plan 03). |
</threat_model>
