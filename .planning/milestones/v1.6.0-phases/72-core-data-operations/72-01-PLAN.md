Create `sdk/python/chromatindb/_codec.py` with these functions. Use `struct.pack`/`struct.unpack` for binary fields, `flatbuffers.Builder` for blob encoding. All multi-byte wire integers are big-endian (`>` format).

Functions to implement:

1. `TOMBSTONE_MAGIC = b"\xDE\xAD\xBE\xEF"` (module constant)

2. `make_tombstone_data(target_hash: bytes) -> bytes`:
   - Validate `len(target_hash) == 32`, raise ValueError if not
   - Return `TOMBSTONE_MAGIC + target_hash` (36 bytes)

3. `encode_blob_payload(namespace_id: bytes, pubkey: bytes, data: bytes, ttl: int, timestamp: int, signature: bytes) -> bytes`:
   - Build FlatBuffer Blob using blob_generated.py builder functions
   - `builder = flatbuffers.Builder(len(data) + 8192)`
   - `builder.ForceDefaults(True)` -- CRITICAL for ttl=0 tombstones
   - Create byte vectors BEFORE BlobStart: ns_vec, pk_vec, dt_vec, sg_vec via `builder.CreateByteVector()`
   - Call BlobStart, BlobAdd*, BlobEnd, builder.Finish, return `bytes(builder.Output())`

4. `decode_write_ack(payload: bytes) -> tuple[bytes, int, bool]`:
   - Validate `len(payload) == 41`, raise ProtocolError if not
   - `blob_hash = payload[:32]`
   - `seq_num = struct.unpack(">Q", payload[32:40])[0]`
   - `duplicate = payload[40] == 1`
   - Return `(blob_hash, seq_num, duplicate)`

5. `decode_delete_ack(payload: bytes) -> tuple[bytes, int, bool]`:
   - Identical format to decode_write_ack (same 41-byte layout)
   - Validate `len(payload) == 41`, raise ProtocolError if not
   - Same parsing as decode_write_ack

6. `encode_read_request(namespace: bytes, blob_hash: bytes) -> bytes`:
   - Validate `len(namespace) == 32` and `len(blob_hash) == 32`, raise ValueError if not
   - Return `namespace + blob_hash` (64 bytes)

7. `decode_read_response(payload: bytes) -> tuple[bytes, int, int, bytes] | None`:
   - If `len(payload) < 1`, raise ProtocolError("empty ReadResponse")
   - If `payload[0] == 0x00`, return None (not found)
   - If `payload[0] != 0x01`, raise ProtocolError
   - Slice `fb_bytes = payload[1:]`, decode via `Blob.GetRootAs(fb_bytes, 0)`
   - Extract data via per-element loop: `bytes(blob.Data(j) for j in range(blob.DataLength()))`
   - Extract ttl via `blob.Ttl()`, timestamp via `blob.Timestamp()`
   - Extract signature via per-element loop same as data
   - Return `(data, ttl, timestamp, signature)`

8. `encode_list_request(namespace: bytes, since_seq: int, limit: int) -> bytes`:
   - Validate `len(namespace) == 32`, raise ValueError if not
   - Return `namespace + struct.pack(">Q", since_seq) + struct.pack(">I", limit)` (44 bytes)

9. `decode_list_response(payload: bytes) -> tuple[list[tuple[bytes, int]], bool]`:
   - If `len(payload) < 5`, raise ProtocolError
   - `count = struct.unpack(">I", payload[:4])[0]`
   - `expected_len = 4 + count * 40 + 1`
   - If `len(payload) != expected_len`, raise ProtocolError
   - Parse entries: for each i in range(count), `off = 4 + i * 40`, hash = payload[off:off+32], seq = struct.unpack(">Q", payload[off+32:off+40])[0]
   - `has_more = payload[-1] == 1`
   - Return `(entries, has_more)`

10. `encode_exists_request(namespace: bytes, blob_hash: bytes) -> bytes`:
    - Validate `len(namespace) == 32` and `len(blob_hash) == 32`, raise ValueError if not
    - Return `namespace + blob_hash` (64 bytes)

11. `decode_exists_response(payload: bytes) -> tuple[bool, bytes]`:
    - If `len(payload) != 33`, raise ProtocolError
    - `exists = payload[0] == 1`
    - `blob_hash = payload[1:33]`
    - Return `(exists, blob_hash)`
