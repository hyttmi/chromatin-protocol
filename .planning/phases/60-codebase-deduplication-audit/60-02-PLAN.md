Path = `std::filesystem::temp_directory_path() / (prefix + std::to_string(random))`.     2. **run_async template** -- `template <typename T> T run_async(asio::thread_pool& pool, asio::awaitable<T> aw)`. Creates local `asio::io_context ioc`. Declares `T result{}`. Spawns coroutine via `asio::co_spawn(ioc, lambda, asio::detached)` where lambda moves the awaitable, co_awaits it into result, co_returns result. Calls `ioc.run()`. Returns result. This is identical to all 4 existing copies.

    3. **current_timestamp** -- `inline uint64_t current_timestamp()`. Returns `static_cast<uint64_t>(std::chrono::duration_cast<std::chrono::seconds>(std::chrono::system_clock::now().time_since_epoch()).count())`.

    4. **TS_AUTO** -- `constexpr uint64_t TS_AUTO = UINT64_MAX;` -- sentinel value meaning "use current time".

    5. **make_signed_blob** -- `inline wire::BlobData make_signed_blob(const identity::NodeIdentity& id, const std::string& payload, uint32_t ttl = 604800, uint64_t timestamp = TS_AUTO)`. Implementation:
       - Create `wire::BlobData blob`
       - `std::memcpy(blob.namespace_id.data(), id.namespace_id().data(), 32)`
       - `blob.pubkey.assign(id.public_key().begin(), id.public_key().end())`
       - `blob.data.assign(payload.begin(), payload.end())`
       - `blob.ttl = ttl`
       - `blob.timestamp = (timestamp == TS_AUTO) ? current_timestamp() : timestamp`
       - `auto signing_input = wire::build_signing_input(blob.namespace_id, blob.data, blob.ttl, blob.timestamp)`
       - `blob.signature = id.sign(signing_input)`
       - Return blob
       - NOTE: Uses TS_AUTO default (current time), NOT the old test_daemon.cpp default of 1000.

    6. **make_signed_tombstone** -- `inline wire::BlobData make_signed_tombstone(const identity::NodeIdentity& id, const std::array<uint8_t, 32>& target_hash, uint64_t timestamp = TS_AUTO)`. Implementation:
       - Create `wire::BlobData tombstone`
       - `std::memcpy(tombstone.namespace_id.data(), id.namespace_id().data(), 32)`
       - `tombstone.pubkey.assign(id.public_key().begin(), id.public_key().end())`
       - Build tombstone data: 4-byte magic `{0xDE, 0xAD, 0xBE, 0xEF}` + 32-byte target_hash
       - `tombstone.data.reserve(36); tombstone.data.insert(tombstone.data.end(), magic, magic+4); tombstone.data.insert(tombstone.data.end(), target_hash.begin(), target_hash.end())`
       - `tombstone.ttl = 0` (permanent)
       - `tombstone.timestamp = (timestamp == TS_AUTO) ? current_timestamp() : timestamp`
       - Build signing input, sign, return

    7. **make_signed_delegation** -- `inline wire::BlobData make_signed_delegation(const identity::NodeIdentity& owner, const identity::NodeIdentity& delegate)`. Implementation:
       - Create `wire::BlobData blob`
       - Copy owner's namespace_id and pubkey
       - Build delegation data: 4-byte magic `{0xDE, 0x1E, 0x6A, 0x7E}` + 32 bytes of SHA3-256(delegate_pubkey). Use `db/crypto/hash.h` `chromatindb::crypto::sha3_256()` for this (add `#include "db/crypto/hash.h"` to the header).
       - Read the exact implementation from test_engine.cpp lines 123-143 to get the delegation data construction correct
       - `blob.ttl = 0` (permanent delegation)
       - `blob.timestamp = current_timestamp()`
       - Build signing input, sign with owner's key, return

    8. **make_delegate_blob** -- `inline wire::BlobData make_delegate_blob(const identity::NodeIdentity& owner, const identity::NodeIdentity& delegate, const std::string& payload, uint32_t ttl = 604800)`. Implementation:
       - Create `wire::BlobData blob`
       - Copy OWNER's namespace_id (delegate writes to owner's namespace)
       - Copy DELEGATE's pubkey
       - `blob.data.assign(payload.begin(), payload.end())`
       - `blob.ttl = ttl`
       - `blob.timestamp = current_timestamp()`
       - Build signing input, sign with DELEGATE's key, return
       - Read the exact implementation from test_engine.cpp lines 144-165 to get this right
