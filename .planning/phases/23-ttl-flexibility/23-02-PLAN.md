       if (wire::is_tombstone(blob.data)) {

    5. **db/storage/storage.cpp** — run_expiry_scan(): Restructure the scan loop to read blob data BEFORE deleting. The critical change is:
       - Read the blob from blobs_map using the hash and namespace from the expiry entry
       - Check if it's a tombstone using wire::is_tombstone()
       - If tombstone: clean tombstone_map entry using wire::extract_tombstone_target()
       - Then delete from blobs_map (existing behavior)
       - Then erase the expiry entry (existing behavior)

       The reordered scan body should be:
       ```cpp
       // Read blob to check if it's a tombstone (must read BEFORE delete)
       auto blob_key = make_blob_key(ns_ptr, hash_ptr);
       auto blob_data = txn.get(impl_->blobs_map, to_slice(blob_key), not_found_sentinel);
       if (blob_data.data() != nullptr) {
           // Check if this is a tombstone and clean tombstone_map
           auto decoded = wire::decode_blob(std::span<const uint8_t>(
               static_cast<const uint8_t*>(blob_data.data()), blob_data.length()));
           if (wire::is_tombstone(decoded.data)) {
               auto target_hash = wire::extract_tombstone_target(
                   std::span<const uint8_t>(decoded.data));
               auto ts_key = make_blob_key(ns_ptr, target_hash.data());
               try { txn.erase(impl_->tombstone_map, to_slice(ts_key)); }
               catch (const mdbx::exception&) { /* already cleaned */ }
           }
           // Delete from blobs_map
           try { txn.erase(impl_->blobs_map, to_slice(blob_key)); }
           catch (const mdbx::exception&) { /* already deleted */ }
       }
       ```

       **CRITICAL:** The existing code does `txn.erase(impl_->blobs_map, to_slice(blob_key))` first and then the cursor operations. The new code must do the read, tombstone check, tombstone_map cleanup, THEN the blobs_map delete.
