<objective>
Fix resource limit enforcement bugs: subscription limits, bootstrap detection, TOCTOU capacity/quota race, and quota rebuild iterator.

Purpose: Four resource limit bugs found in the v2.2.0 audit: (1) no subscription limit allows memory exhaustion, (2) bootstrap detection ignores port so multi-node hosts break, (3) capacity/quota checks race with concurrent ingests across co_await, (4) quota rebuild skips every other entry during clear.

Output: All four bugs fixed with unit tests. store_blob performs atomic capacity/quota check inside its write transaction. Subscription limit configurable via config.
</objective>
