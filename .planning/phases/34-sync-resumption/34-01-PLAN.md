    **Storage additions (storage.h):**
    1. Add `SyncCursor` struct with three fields: `uint64_t seq_num = 0`, `uint32_t round_count = 0`, `uint64_t last_sync_timestamp = 0`
    2. Add 7 public methods to Storage class:
       - `std::optional<SyncCursor> get_sync_cursor(std::span<const uint8_t, 32> peer_hash, std::span<const uint8_t, 32> namespace_id)`
       - `void set_sync_cursor(std::span<const uint8_t, 32> peer_hash, std::span<const uint8_t, 32> namespace_id, const SyncCursor& cursor)`
       - `void delete_sync_cursor(std::span<const uint8_t, 32> peer_hash, std::span<const uint8_t, 32> namespace_id)`
       - `size_t delete_peer_cursors(std::span<const uint8_t, 32> peer_hash)` -- delete all cursors for a peer
       - `size_t reset_all_round_counters()` -- set round_count=0 for all cursors (SIGHUP)
       - `std::vector<std::array<uint8_t, 32>> list_cursor_peers()` -- unique peer hashes
       - `size_t cleanup_stale_cursors(const std::vector<std::array<uint8_t, 32>>& known_peer_hashes)` -- delete cursors for peers NOT in the known set, return count deleted

    **Storage implementation (storage.cpp):**
    1. Add `mdbx::map_handle cursor_map{0}` to Impl struct
    2. Increase `max_maps` from 6 to 7 (6 sub-databases + 1 spare)
    3. Add `cursor_map = txn.create_map("cursor")` alongside existing 5 sub-dbs
    4. Add `make_cursor_key(peer_hash, ns)` returning `std::array<uint8_t, 64>` -- same pattern as make_blob_key
    5. Add cursor value encoding/decoding: `[seq_num_be:8][round_count_be:4][last_sync_ts_be:8]` = 20 bytes total. Use existing `encode_be_u64`/`decode_be_u64` for 8-byte fields. Manual 4-byte big-endian for round_count (same shift pattern).
    6. CRUD methods use single write transactions each -- no read transaction held across co_await (safe by design since Storage is synchronous)
    7. `delete_peer_cursors`: open cursor on cursor_map, seek to `[peer_hash:32][0x00*32]`, iterate while first 32 bytes match peer_hash, delete each
    8. `reset_all_round_counters`: iterate all cursor entries, decode value, zero out round_count bytes (positions 8-11), write back. Return count.
    9. `list_cursor_peers`: iterate all entries, collect unique first-32-bytes from keys into a set, return as vector
    10. `cleanup_stale_cursors`: call list_cursor_peers(), for each peer_hash NOT in known_peer_hashes set, call delete_peer_cursors(). Return total deleted count.
