<!-- UDS handshake protocol (from relay/core/uds_multiplexer.cpp) -->
UDS connection sequence:
1. Connect to UDS socket path
2. Send TrustedHello: [magic:4 "CHDB"][version:1][suite:1]
3. Receive TrustedHello response
4. HKDF key derivation from shared secret
5. Auth exchange (challenge-response using relay identity)
6. Now can send/receive AEAD-encrypted TransportMessages

<!-- Smoke test WebSocket protocol -->
WebSocket connection sequence:
1. TCP connect to relay host:port (or TLS connect)
2. HTTP upgrade handshake
3. Receive auth challenge JSON: {"type":"auth_challenge","challenge":"<hex>"}
4. Sign challenge with ML-DSA-87, send auth response JSON
5. Receive auth result: {"type":"auth_result","status":"ok"}
6. Now can send/receive JSON messages
