NodeInfoResponse(40) — node sends:
  [version_len:u8][version:N bytes]
  [git_hash_len:u8][git_hash:N bytes]
  [uptime:u64BE][peer_count:u32BE][namespace_count:u32BE]
  [total_blobs:u64BE][storage_used:u64BE][storage_max:u64BE]
  [types_count:u8][type_bytes:types_count raw bytes]
Note: string lengths are u8 (NOT u16BE), supported_types are raw bytes (NOT length-prefixed strings)

StatsResponse(36) — node sends:
  [blob_count:u64BE][total_bytes:u64BE][byte_limit:u64BE]
Total: 24 bytes. Per-namespace stats (StatsRequest takes a 32-byte namespace).

StatsRequest(35) — node expects:
  [namespace:32 bytes]
Note: relay schema currently MISSING the namespace field (only has request_id).

TimeRangeResponse(58) — node sends:
  [truncated:u8][count:u32BE][ [hash:32][seq_num:u64BE][timestamp:u64BE] * count ]
Note: truncated is FIRST (not last), entries are 48 bytes (not 40 — includes seq_num)

DelegationListResponse(52) — node sends:
  [count:u32BE][ [delegate_pk_hash:32][delegation_blob_hash:32] * count ]
Note: binary format correct, JSON field names wrong (namespace/pubkey_hash should be delegate_pk_hash/delegation_blob_hash)

    ```cpp
    static std::optional<nlohmann::json> decode_stats_response(std::span<const uint8_t> p) {
        if (p.size() < 24) return std::nullopt;
        nlohmann::json j;
        j["type"] = "stats_response";
        size_t off = 0;
        j["blob_count"] = std::to_string(util::read_u64_be(p.data() + off)); off += 8;
        j["storage_bytes"] = std::to_string(util::read_u64_be(p.data() + off)); off += 8;
        j["quota_bytes_limit"] = std::to_string(util::read_u64_be(p.data() + off)); off += 8;
        return j;
    }
    ```
