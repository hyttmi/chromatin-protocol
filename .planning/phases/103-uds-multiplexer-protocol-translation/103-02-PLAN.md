From db/net/connection.cpp (reference for TrustedHello handshake -- relay reimplements):
```
Initiator handshake:
1. Generate 32-byte nonce, build payload [nonce:32][signing_pubkey:2592]
2. Send raw: TransportCodec::encode(TrustedHello, payload) wrapped in [4B BE len]
3. Recv raw: [4B BE len][TransportMessage] -- expect TrustedHello response
4. Parse response nonce + signing_pk
5. derive_lightweight_session_keys(nonce_i, nonce_r, pk_i, pk_r, is_initiator=true)
6. Send encrypted: TransportCodec::encode(AuthSignature, auth_payload)
   auth_payload = [pubkey_size:4B LE][pubkey:2592][signature:4627]
   signature = identity.sign(session_fingerprint)
7. Recv encrypted: AuthSignature from node -- verify signature on session_fingerprint
8. After auth: send_counter_=1, recv_counter_=1 (counter 0 consumed by auth exchange)
```

                       RequestRouter& router,
                       ws::SessionManager& sessions);

        void start();

        /// Returns false if not connected.
        bool send(std::vector<uint8_t> transport_msg);

    private:

        asio::awaitable<void> connect_loop();

        /// Perform TrustedHello + HKDF + auth exchange.
        asio::awaitable<bool> do_handshake();

        /// Post-handshake read loop: recv_encrypted -> decode -> route response.
        asio::awaitable<void> read_loop();

        /// Periodic stale request cleanup (60s sweep). Per D-11.
        asio::awaitable<void> cleanup_loop();

        /// Send raw frame: [4B BE length][data]. No encryption (for handshake).
        asio::awaitable<bool> send_raw(std::span<const uint8_t> data);

        /// Recv raw frame: read [4B BE length], then read [data]. No encryption.
        asio::awaitable<std::optional<std::vector<uint8_t>>> recv_raw();

        asio::awaitable<bool> send_encrypted(std::span<const uint8_t> plaintext);

        /// Recv encrypted frame: recv_raw, then AEAD decrypt.
        asio::awaitable<std::optional<std::vector<uint8_t>>> recv_encrypted();

        void route_response(uint8_t type, std::vector<uint8_t> payload, uint32_t request_id);

        std::string uds_path_;
        const identity::RelayIdentity& identity_;
        RequestRouter& router_;
        ws::SessionManager& sessions_;

        asio::local::stream_protocol::socket socket_;
        bool connected_ = false;

        // AEAD state
        std::vector<uint8_t> send_key_;   // 32 bytes
        std::vector<uint8_t> recv_key_;   // 32 bytes

        uint64_t recv_counter_ = 0;

    **3. Create relay/core/uds_multiplexer.cpp**:

    **connect_loop():**
    - Loop forever with try/catch:
      - Create socket as `asio::local::stream_protocol::socket(ioc_)`
      - `co_await socket_.async_connect(endpoint, asio::use_awaitable)`
      - On success: log info, call `do_handshake()`
      - If handshake succeeds: set `connected_ = true`, log info, spawn `read_loop()` and `cleanup_loop()`, co_return
      - On connect failure or handshake failure: close socket, compute jittered backoff (base 1s, cap 30s, jitter = random 0-50% of delay), sleep, retry
    - Backoff formula: `delay = min(base * 2^attempt, 30s)`, jitter = `delay * (0.5 + random(0.0, 0.5))`
    - Use `asio::steady_timer` for sleep

    **do_handshake():**
    Follow the node's initiator path exactly (reference: db/net/connection.cpp lines 265-363):

    Step 1: Generate 32-byte nonce via `RAND_bytes(nonce.data(), 32)` (OpenSSL)
    Step 2: Build TrustedHello payload: `[nonce:32][signing_pubkey:2592]`
    Step 3: Encode with TransportCodec: `TransportCodec::encode(TransportMsgType_TrustedHello, payload)`
    Step 4: `send_raw(encoded)` -- write `[4B BE length][encoded_data]`
    Step 5: `recv_raw()` -- read `[4B BE length][data]`
    Step 6: Decode response with TransportCodec. Verify type is TrustedHello.
    Step 7: Parse response: `nonce_r` (32 bytes) + `resp_signing_pk` (2592 bytes)
    Step 8: Derive session keys via HKDF-SHA256:
      - IKM = `nonce_i || nonce_r` (64 bytes)
      - Salt = `signing_pk_i || signing_pk_r` (5184 bytes) -- relay is initiator
      - PRK = `hkdf_extract(salt, ikm)`
      - init_to_resp_key = `hkdf_expand(prk, "chromatin-init-to-resp-v1", 32)` -- relay's send key
      - resp_to_init_key = `hkdf_expand(prk, "chromatin-resp-to-init-v1", 32)` -- relay's recv key
      - session_fingerprint = SHA3-256(ikm || salt) -- use `OQS_SHA3_sha3_256`
    Step 9: Set `send_key_ = init_to_resp_key`, `recv_key_ = resp_to_init_key`
    Step 10: Auth exchange (encrypted, consumes counter 0):
      - Sign session fingerprint: `auto sig = identity_.sign(session_fingerprint)`
      - Build auth payload: `[pubkey_size:4B LE][pubkey:2592][signature:4627]`
        CRITICAL: pubkey_size is LITTLE-endian (protocol exception per project memory)
        `uint32_t pk_size = 2592; payload[0] = pk_size & 0xFF; payload[1] = (pk_size >> 8) & 0xFF; ...`
      - Encode: `TransportCodec::encode(TransportMsgType_AuthSignature, auth_payload)`
      - `send_encrypted(encoded)` -- uses counter 0, increments to 1
      - `recv_encrypted()` -- uses counter 0, increments to 1
      - Decode response: verify type is AuthSignature
      - Parse auth payload: read LE pubkey_size, extract pubkey + signature
      - Verify signature on session_fingerprint using `OQS_SIG_verify` (ML-DSA-87)
      - Verify pubkey matches the one from TrustedHello response
    Step 11: Post-handshake: `send_counter_` and `recv_counter_` are now both 1.

    **send_raw(data):**
    - Build header: `[4B BE length of data]`
    - Write header + data to socket using `asio::async_write`

    **recv_raw():**
    - Read 4-byte header from socket
    - Parse BE length, validate <= 110 MiB (MAX_FRAME_SIZE from db/net/framing.h)
    - Read `length` bytes from socket
    - Return vector

    **send_encrypted(plaintext):**
    - `auto ct = aead_encrypt(plaintext, send_key_, send_counter_++)`
    - `send_raw(ct)`

    **recv_encrypted():**
    - `auto raw = recv_raw()`
    - `auto pt = aead_decrypt(*raw, recv_key_, recv_counter_++)`
    - Return plaintext or nullopt

    **send(transport_msg):**
    - If not connected_, return false
    - `co_spawn(ioc_, send_encrypted(transport_msg), detached)` -- fire-and-forget
    - Return true
    - NOTE: For thread safety, use `asio::post(ioc_, ...)` to serialize. The send must go through the strand/io_context. Use a send queue + drain coroutine pattern matching core::Session:
      - `send()` pushes to `send_queue_`, if not draining_, spawns drain coroutine
      - drain coroutine pops from front, calls `send_encrypted()`, loops until empty

    **read_loop():**
    - Loop: `recv_encrypted()` -> `TransportCodec::decode()` -> `route_response(type, payload, request_id)`
    - On recv failure: set `connected_ = false`, close socket, log error, spawn `connect_loop()` again for reconnect (basic retry, NOT the Phase 104 full reconnect with subscription replay)

    **route_response(type, payload, request_id):**
    - If request_id == 0: this is a server-initiated message (Notification, StorageFull, QuotaExceeded). For Phase 103, log and drop (Phase 104 handles Notification fan-out). StorageFull/QuotaExceeded with request_id=0 can be broadcast to all sessions.
    - If request_id != 0: call `router_.resolve_response(request_id)`. If found:
      - Call `binary_to_json(type, payload)` from translator (imported from relay/translate/translator.h)
      - Restore client's original request_id in the JSON
      - Get session via `sessions_.get_session(pending.client_session_id)`
      - If `is_binary_response(type)`: send as binary WS frame (per PROT-04)
      - Else: send as text WS frame via `send_json()`

    **cleanup_loop():**
    - Every 60 seconds: `router_.purge_stale()`
    - Use `asio::steady_timer`

    2. Look up `schema_for_name()` to get MessageSchema.

    4. If `is_flatbuffer && wire_type != 8`: error -- clients don't send ReadResponse or BatchReadResponse
    5. For non-FlatBuffer types: iterate FieldSpec array, encode each field:
