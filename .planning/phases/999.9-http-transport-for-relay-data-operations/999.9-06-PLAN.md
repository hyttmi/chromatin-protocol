    **relay/http/handlers_query.h**: QueryHandlers class holding references to UdsMultiplexer, RequestRouter, ResponsePromiseMap, and config atomics.

    All query handlers follow the same pattern:
    1. Build JSON object from HTTP request (path params, query params, or JSON body)
    2. Call json_to_binary(json) to get wire_type + binary payload
    3. Register request, create ResponsePromise, encode via TransportCodec, send via UDS
    4. co_await promise.wait(timeout)
    5. binary_to_json(response_type, response_payload) -> JSON
    6. Return HttpResponse::json(200, json)

    Create a private helper: `asio::awaitable<HttpResponse> forward_query(const nlohmann::json& query_json, uint64_t session_id)` that encapsulates steps 2-6.
