```python
from chromatindb import Identity

# Ephemeral (in-memory only)
identity = Identity.generate()

identity = Identity.generate_and_save("my_identity.key")

# Load existing
identity = Identity.load("my_identity.key")
```
Briefly explain: identity = ML-DSA-87 keypair, namespace = SHA3-256(public_key).

**Section 3: Connect to a Relay**
```python
import asyncio
from chromatindb import ChromatinClient, Identity

async def main():
    identity = Identity.generate()

    async with ChromatinClient.connect("192.168.1.200", 4201, identity) as client:
        await client.ping()
        print("Connected!")

asyncio.run(main())
```
Explain: PQ handshake happens automatically, connection is AEAD-encrypted, context manager sends Goodbye on exit.

**Section 4: Write and Read a Blob**
```python
async with ChromatinClient.connect("192.168.1.200", 4201, identity) as client:
    # Write a blob with 1-hour TTL
    result = await client.write_blob(b"Hello, chromatindb!", ttl=3600)
    print(f"Written: hash={result.blob_hash.hex()}, seq={result.seq_num}")

    # Read it back using our own namespace
    read = await client.read_blob(identity.namespace, result.blob_hash)
    if read is not None:
        print(f"Read: {read.data}")  # b"Hello, chromatindb!"
```
Explain: write_blob signs with your identity, TTL is in seconds, read_blob takes namespace + hash, returns None if not found.
