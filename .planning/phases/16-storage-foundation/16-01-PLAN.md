delegation_map = txn.create_map("delegation");

// Index write in store_blob():
if (wire::is_delegation(blob.data)) {
    auto delegate_pubkey = wire::extract_delegate_pubkey(blob.data);
    auto delegate_pk_hash = crypto::sha3_256(delegate_pubkey);
    auto deleg_key = make_blob_key(blob.namespace_id.data(), delegate_pk_hash.data());
    txn.upsert(impl_->delegation_map, to_slice(deleg_key),
                mdbx::slice(hash.data(), hash.size()));
}

// Index cleanup in delete_blob_data():
if (wire::is_delegation(blob.data)) {
    auto delegate_pubkey = wire::extract_delegate_pubkey(blob.data);
    auto delegate_pk_hash = crypto::sha3_256(delegate_pubkey);
    auto deleg_key = make_blob_key(ns.data(), delegate_pk_hash.data());
    try {
        txn.erase(impl_->delegation_map, to_slice(deleg_key));
    } catch (const mdbx::exception&) { }
}

// O(1) lookup in has_valid_delegation():
auto deleg_key = make_blob_key(namespace_id.data(), delegate_pk_hash.data());

auto val = txn.get(impl_->delegation_map, to_slice(deleg_key), not_found_sentinel);

    **storage.h changes:**
    1. Add `uint64_t used_bytes() const;` public method declaration after `run_expiry_scan()`.
    2. Update the class docstring to mention 5 sub-databases (blobs, sequence, expiry, delegation, tombstone).

    **storage.cpp changes (follow delegation_map pattern exactly):**

    1. **Impl struct:** Add `mdbx::map_handle tombstone_map;` member alongside `delegation_map`.

    2. **Impl constructor:** Bump `max_maps` from 5 to 6 (5 sub-databases + 1 spare). Create the tombstone sub-database in the same write transaction as the others:
       ```cpp
       tombstone_map = txn.create_map("tombstone");
       ```

    3. **Startup migration:** After the write transaction commits and sub-databases are created, perform a one-time migration to populate `tombstone_map` from existing tombstone blobs:
       - Open a read transaction, scan `blobs_map` with a cursor
       - Collect entries where `wire::is_tombstone(blob.data)` is true: store (namespace, target_hash) pairs
       - Batch-write to `tombstone_map` in write transactions of ~1000 entries each
       - Log migration count at `info` level: "Migrated {} existing tombstones to tombstone index"
       - If count is 0, skip logging (no noise for fresh databases)

    4. **store_blob():** After the existing delegation_map indexing block, add a parallel block for tombstone indexing:
       ```cpp
       if (wire::is_tombstone(blob.data)) {
           auto target_hash = wire::extract_tombstone_target(
               std::span<const uint8_t>(blob.data));
           auto ts_key = make_blob_key(blob.namespace_id.data(), target_hash.data());
           txn.upsert(impl_->tombstone_map, to_slice(ts_key), mdbx::slice());
       }
       ```
       Key: `[namespace:32][target_hash:32]`, Value: empty (existence check only).

    5. **has_tombstone_for():** Replace the entire O(n) cursor scan body with an O(1) lookup:
       ```cpp
       auto ts_key = make_blob_key(ns.data(), target_blob_hash.data());
       auto txn = impl_->env.start_read();
       auto val = txn.get(impl_->tombstone_map, to_slice(ts_key), not_found_sentinel);
       return val.data() != nullptr;
       ```
       Remove all cursor code, the loop, and the is_tombstone/extract_tombstone_target calls from within this method.

    6. **delete_blob_data():** After the existing delegation_map cleanup block, add tombstone_map cleanup. Follow the same guard pattern -- only attempt cleanup when the deleted blob is a tombstone:
       ```cpp
       if (wire::is_tombstone(blob.data)) {
           auto target_hash = wire::extract_tombstone_target(
               std::span<const uint8_t>(blob.data));
           auto ts_key = make_blob_key(ns.data(), target_hash.data());
           try {
               txn.erase(impl_->tombstone_map, to_slice(ts_key));
           } catch (const mdbx::exception&) {
               // Already deleted -- not an error
           }
       }
       ```
       NOTE: The blob data must be read BEFORE deleting from blobs_map, because we need to check is_tombstone(). Verify the existing code reads the blob data before the erase call. If not, read it first.

    7. **used_bytes():** Implement the new method:
       ```cpp
       uint64_t Storage::used_bytes() const {
           auto info = impl_->env.get_info();
           return info.mi_geo.current;
       }
       ```
       This returns the actual mdbx database file size in bytes. It is O(1), authoritative, and does not require scanning.
