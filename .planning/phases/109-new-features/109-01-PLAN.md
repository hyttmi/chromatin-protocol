- `using BlobHash32 = std::array<uint8_t, 32>;`
- Include `relay/core/subscription_tracker.h` for `Namespace32Hash` reuse (works for any 32-byte SHA3-256 output)
- `struct Entry { uint64_t session_id; std::chrono::steady_clock::time_point created; };`
- `static constexpr auto TTL = std::chrono::seconds(5);` (per D-04)
- `std::unordered_map<BlobHash32, Entry, Namespace32Hash> entries_;`
- `void record(const BlobHash32& blob_hash, uint64_t session_id)` -- lazy expiry sweep on each call (iterate entries, erase older than TTL), then insert/overwrite
- `std::optional<uint64_t> lookup_and_remove(const BlobHash32& blob_hash)` -- find, check expiry, erase, return session_id or nullopt
- `void remove_session(uint64_t session_id)` -- iterate and erase all entries for that session
- `size_t size() const` -- return entries_.size()
- Add comment at top: `/// NOT thread-safe -- accessed only from UDS read_loop coroutine.`
