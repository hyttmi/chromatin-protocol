       **Constants** (per D-06, D-07, Pitfall 1):
       ```python
       USERENTRY_MAGIC = b"UENT"
       USERENTRY_VERSION = 0x01
       DELEGATION_MAGIC = bytes([0xDE, 0x1E, 0x6A, 0x7E])
       # Minimum: magic(4) + ver(1) + signing_pk(2592) + kem_pk(1568) + name_len(2) + kem_sig(>=1)
       USERENTRY_MIN_SIZE = 4 + 1 + PUBLIC_KEY_SIZE + KEM_PUBLIC_KEY_SIZE + 2 + 1
       ```

       **DirectoryEntry** (per D-19): `@dataclass(frozen=True)` with `identity: Identity`, `display_name: str`, `blob_hash: bytes`. Place at module top after constants.

       **encode_user_entry(identity: Identity, display_name: str) -> bytes** (per D-06, D-08, D-09, D-26):
       - Validate identity.has_kem and identity.can_sign (raise ValueError if not)
       - Encode display_name to UTF-8, check len <= 256 bytes
       - Sign KEM pubkey: `kem_sig = identity.sign(identity.kem_public_key)` (D-08)
       - Return concatenation: USERENTRY_MAGIC + struct.pack("B", USERENTRY_VERSION) + identity.public_key + identity.kem_public_key + struct.pack(">H", len(name_bytes)) + name_bytes + kem_sig
       - Import struct, use big-endian per _codec.py convention

       **decode_user_entry(data: bytes) -> tuple[bytes, bytes, str, bytes] | None** (per D-06, Pitfall 3, Pitfall 4):
       - Return None if: len < USERENTRY_MIN_SIZE, magic != USERENTRY_MAGIC, version != USERENTRY_VERSION
       - Parse fixed offsets: signing_pk at [5:5+2592], kem_pk at [2597:2597+1568], name_len from [4165:4167] BE uint16
       - Validate: offset + name_len must not exceed data length
       - display_name = data[4167:4167+name_len].decode("utf-8")
       - kem_sig = data[4167+name_len:] (remainder -- Pitfall 4: do NOT add length prefix for kem_sig)
       - Return None if kem_sig is empty

       **verify_user_entry(signing_pk: bytes, kem_pk: bytes, kem_sig: bytes) -> bool** (per D-08, D-25):
       - Return `Identity.verify(kem_pk, kem_sig, signing_pk)` -- verifies that signing_pk signed kem_pk

       **make_delegation_data(delegate_signing_pk: bytes) -> bytes** (per Pitfall 1, PROTOCOL.md):
       - Validate len(delegate_signing_pk) == PUBLIC_KEY_SIZE (2592), raise ValueError if not
       - Return DELEGATION_MAGIC + delegate_signing_pk (total 2596 bytes)
