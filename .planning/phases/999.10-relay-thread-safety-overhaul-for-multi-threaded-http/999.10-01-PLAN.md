<objective>
Create the global strand in relay_main, wire it into UdsMultiplexer so all UDS coroutines run strand-confined, and remove all partial mutex fixes from the 5 shared data structures.

Purpose: This is the foundation of the strand confinement model. The strand replaces all mutexes by serializing access to shared state. UDS coroutines (connect_loop, read_loop, cleanup_loop, drain_send_queue) are the primary consumers of shared state and must be the first to migrate. After this plan, shared state is only safe to access from the strand -- Plan 02 will update HTTP handlers to post to strand.

Output: UdsMultiplexer accepts a strand, all its coroutines run on it, all 5 shared data structures are mutex-free.
</objective>
