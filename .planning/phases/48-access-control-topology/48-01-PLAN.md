**Loadgen --delegate flag:**

1. Add `std::string delegate_pubkey_hex;` to the Config struct (empty = not delegation mode).
2. Add CLI parsing for `--delegate PUBKEY_HEX` (a hex string of the delegate's ML-DSA-87 public key, 5184 hex chars = 2592 bytes).
3. Add help text: `"  --delegate PUBKEY_HEX  Create a delegation blob for the given delegate public key\n"`.
4. Create a `make_delegation_blob` function near the existing `make_signed_blob` and `make_tombstone_request`:
   ```cpp
   chromatindb::wire::BlobData make_delegation_blob(
       const chromatindb::identity::NodeIdentity& id,
       std::span<const uint8_t> delegate_pubkey,
       uint32_t ttl, uint64_t timestamp)
   {
       chromatindb::wire::BlobData blob;
       std::memcpy(blob.namespace_id.data(), id.namespace_id().data(), 32);
       blob.pubkey.assign(id.public_key().begin(), id.public_key().end());
       blob.data = chromatindb::wire::make_delegation_data(delegate_pubkey);
       blob.ttl = ttl;
       blob.timestamp = timestamp;
       auto signing_input = chromatindb::wire::build_signing_input(
           blob.namespace_id, blob.data, blob.ttl, blob.timestamp);
       blob.signature = id.sign(signing_input);
       return blob;
   }
   ```
5. In the `subscribe_and_send` coroutine, add a delegation path. When `cfg_.delegate_pubkey_hex` is non-empty:
   - Decode the hex pubkey to bytes using a `from_hex_bytes` function (variable length, unlike the 32-byte `from_hex`). Add a helper that decodes arbitrary-length hex to `std::vector<uint8_t>`.
   - Validate the decoded pubkey is exactly 2592 bytes (DELEGATION_PUBKEY_SIZE). Log error and abort if not.
   - For each blob to send (count defaults to 1 for delegation), construct via `make_delegation_blob` instead of `make_signed_blob`.
   - The blob is sent via `TransportMsgType_Data` (same as normal blobs -- the server identifies delegation blobs by their DELEGATION_MAGIC prefix in data).
   - Set `cfg_.count` to 1 if user didn't specify `--count` explicitly and `--delegate` is set (one delegation blob is typical). Actually, just document that `--count 1` should be used; don't override.
6. The `--delegate` flag is independent of `--delete` (they should not be combined). Add a check: if both are set, print error and exit.
