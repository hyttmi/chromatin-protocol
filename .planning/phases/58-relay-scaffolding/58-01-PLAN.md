    - `log_level`: must be one of trace/debug/info/warn/warning/error/err/critical
    - `log_file`: no validation (empty = console only)

    JSON field mapping: `bind_address`, `bind_port`, `uds_path`, `identity_key_path`, `log_level`, `log_file`. Warn on unknown keys (same pattern as node config).
