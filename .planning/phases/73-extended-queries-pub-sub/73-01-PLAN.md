    Key implementation details:

    **encode_metadata_request:** Same pattern as encode_read_request. Returns namespace + blob_hash (64 bytes). Validate both are 32 bytes.

    **decode_metadata_response:** Check first byte. 0x00 = return None. 0x01 = parse: blob_hash=payload[1:33], timestamp=struct.unpack(">Q", payload[33:41])[0], ttl=struct.unpack(">I", payload[41:45])[0], data_size=struct.unpack(">Q", payload[45:53])[0], seq_num=struct.unpack(">Q", payload[53:61])[0], pubkey_len=struct.unpack(">H", payload[61:63])[0], pubkey=payload[63:63+pubkey_len]. Validate len(payload)==63+pubkey_len. Return MetadataResult(...).

    **encode_batch_exists_request:** Validate namespace 32 bytes, each hash 32 bytes. Return namespace + struct.pack(">I", count) + b"".join(hashes).

    **decode_batch_exists_response(payload, hashes):** Takes the original hashes list to build the dict. Validate len(payload)==len(hashes). Return {hashes[i]: payload[i]==1 for i in range(len(hashes))}.

    **encode_batch_read_request:** Validate namespace 32 bytes, each hash 32 bytes. Return namespace + struct.pack(">I", cap_bytes) + struct.pack(">I", count) + b"".join(hashes).

    **decode_batch_read_response:** Parse truncated=payload[0]==1, count=struct.unpack(">I", payload[1:5])[0]. Then iterate entries: for each, status=payload[off], hash=payload[off+1:off+33].
