<objective>
Implement the SyncNamespaceAnnounce protocol (message type 62) so peers exchange their replication scope after handshake, BlobNotify is filtered by namespace, and Phase A/B/C reconciliation is scoped to the namespace intersection.

Purpose: Eliminates wasted bandwidth by only sending BlobNotify and reconciliation traffic to peers that actually replicate the relevant namespaces. This is the core protocol change for Phase 86 (FILT-01 + FILT-02).

Output: Modified transport schema with type 62, PeerInfo with announce tracking, announce exchange coroutine, BlobNotify filtering, reconciliation filtering, SIGHUP re-announce, relay blocklist update, unit tests for announce encode/decode and filtering logic.

**Change 2: Implement announce_and_sync coroutine** (per D-01)

Add a new method `PeerManager::announce_and_sync`:
```cpp
asio::awaitable<void> PeerManager::announce_and_sync(net::Connection::Ptr conn) {
    auto* peer = find_peer(conn);
    if (!peer) co_return;

    // Send our sync_namespaces (using existing encode_namespace_list)
    auto ns_list = std::vector<std::array<uint8_t, 32>>(
        sync_namespaces_.begin(), sync_namespaces_.end());
    auto payload = encode_namespace_list(ns_list);
    if (!co_await conn->send_message(
            wire::TransportMsgType_SyncNamespaceAnnounce, payload)) {
        co_return;
    }

    // Wait for peer's announce (timeout 5s) using timer-cancel pattern
    if (!peer->announce_received) {
        asio::steady_timer timer(ioc_);
        peer->announce_notify = &timer;
        timer.expires_after(std::chrono::seconds(5));
        auto [ec] = co_await timer.async_wait(
            asio::as_tuple(asio::use_awaitable));
        peer->announce_notify = nullptr;
        if (!peer->announce_received) {
            spdlog::warn("peer {} did not send SyncNamespaceAnnounce within 5s, treating as replicate-all",
                         peer_display_name(conn));
            // D-07: Treat timeout as "replicate everything" (empty set)
        }
    }

    // Initiator triggers sync after announce exchange (per D-01)
    if (conn->is_initiator()) {
        co_await run_sync_with_peer(conn);
    }
}
```
