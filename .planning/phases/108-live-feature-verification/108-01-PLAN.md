**NEW function -- ws_recv_frame_raw (per RESEARCH.md Pattern 2):**
```cpp
inline std::optional<WsFrame> ws_recv_frame_raw(int fd) {
    // Same frame reading logic as ws_recv_frame (read 2-byte header, extended length,
    // optional mask, payload) but does NOT swallow close frames.
    // On OPCODE_CLOSE: parse close code from first 2 bytes of payload (BE uint16),
    // set WsFrame.close_code, return the frame instead of nullopt.
    // On OPCODE_PING: respond with PONG and recurse (same as ws_recv_frame).
    // On all other opcodes: return WsFrame normally.
}
```
This is critical for the SIGTERM test (E2E-05) which must verify close code 1001.
