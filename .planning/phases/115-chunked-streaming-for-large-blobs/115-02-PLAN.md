**A. Implement `send_chunked()` in `uds_multiplexer.cpp`:**

This method sends a large payload as chunked sub-frames. It is designed to be called AS PART OF the drain_send_queue path — meaning the chunked sequence is atomic from the UDS wire perspective (no other messages interleave).

```cpp
asio::awaitable<bool> UdsMultiplexer::send_chunked(
    uint8_t type, uint32_t request_id,
    std::span<const uint8_t> payload,
    std::span<const uint8_t> extra_metadata) {

    // 1. Send chunked header sub-frame
    auto header = core::encode_chunked_header(type, request_id,
                                               static_cast<uint64_t>(payload.size()),
                                               extra_metadata);
    if (!co_await send_encrypted(header)) co_return false;

    // 2. Send data sub-frames (CHUNK_SIZE each)
    size_t offset = 0;
    while (offset < payload.size()) {
        size_t chunk_len = std::min(core::CHUNK_SIZE, payload.size() - offset);
        auto chunk_span = payload.subspan(offset, chunk_len);
        if (!co_await send_encrypted(chunk_span)) co_return false;
        offset += chunk_len;
    }

    // 3. Send zero-length end sentinel
    if (!co_await send_encrypted(std::span<const uint8_t>{})) co_return false;

    co_return true;
}
```

Key points:
- Each call to send_encrypted() increments send_counter_ and does per-chunk AEAD (D-08, D-09).
- send_encrypted() already uses offload_if_large() with counter-by-value capture (Phase 114 D-10 pattern). Since each chunk is 1 MiB > 64 KiB threshold, all chunks will be offloaded to the thread pool.
- The zero-length sentinel for send_encrypted with empty span: send_encrypted encrypts 0 bytes of plaintext, producing just a 16-byte AEAD tag. send_raw writes [4B length=16][16 bytes tag]. The receiver calls recv_encrypted which gets the 16-byte frame, decrypts it to 0 bytes, returns empty vector — detected as sentinel.

**B. Modify `send()` public API to support chunked sending:**

The current `send()` method takes a pre-encoded TransportMessage and pushes it to send_queue_. For chunked mode, we need a different entry point because we don't use TransportCodec encoding.

Add a new public method:
```cpp
/// Send a large message via chunked sub-frames. The payload is NOT TransportCodec-encoded.
/// The chunked header carries type + request_id (replacing TransportCodec's envelope).
/// Returns false if not connected.
bool send_chunked_async(uint8_t type, uint32_t request_id,

Implementation: check connected_, then co_spawn a coroutine that calls send_chunked(). The chunked send must be serialized with other sends — it must go through the drain mechanism. 

However, the current drain_send_queue pops one message at a time and calls send_encrypted. A chunked send is multiple send_encrypted calls. We need to ensure atomicity.

    uint32_t request_id;
    std::vector<uint8_t> payload;
    std::vector<uint8_t> extra_metadata;
};
```

Add a second deque or use `std::variant<std::vector<uint8_t>, ChunkedSendJob>` for send_queue_ entries.

    while (!send_queue_.empty() && connected_) {

**A. Implement `recv_chunked_reassemble()` in `uds_multiplexer.cpp`:**

This method is called when read_loop() detects a CHUNKED_BEGIN flag in a received frame. It reads subsequent data sub-frames until the zero-length sentinel and reassembles into a complete payload.

```cpp
asio::awaitable<std::optional<std::pair<ChunkedHeader, std::vector<uint8_t>>>>
UdsMultiplexer::recv_chunked_reassemble(std::span<const uint8_t> first_frame_data) {
    // 1. Parse chunked header from first_frame_data
    auto header = core::decode_chunked_header(first_frame_data);
    if (!header) {
        spdlog::error("UDS: invalid chunked header");
        co_return std::nullopt;
    }

    // 2. Validate total size against MAX_BLOB_DATA_SIZE (500 MiB)
    if (header->total_payload_size > 500ULL * 1024 * 1024) {
        spdlog::error("UDS: chunked payload too large: {} bytes", header->total_payload_size);
        co_return std::nullopt;
    }

    // 3. Reassemble data sub-frames
    std::vector<uint8_t> payload;
    payload.reserve(static_cast<size_t>(header->total_payload_size));

    while (true) {
        auto chunk = co_await recv_encrypted();
        if (!chunk) {
            spdlog::error("UDS: chunked recv failed mid-sequence");
            co_return std::nullopt;
        }

        // Zero-length sentinel = end of chunked sequence
        if (chunk->empty()) break;

        payload.insert(payload.end(), chunk->begin(), chunk->end());

        // Safety: accumulated exceeds declared size
        if (payload.size() > static_cast<size_t>(header->total_payload_size) + 64) {
            spdlog::error("UDS: chunked reassembly exceeded declared size");
            co_return std::nullopt;
        }
    }

    co_return std::make_pair(std::move(*header), std::move(payload));
}
```
