### Notable Finding: Dead Config Field

`Config::storage_path` (config.h) is parsed from JSON but never consumed at runtime. `main.cpp` uses `config.data_dir` exclusively for `Storage` construction. A user configuring `storage_path` would have their setting silently ignored. Non-blocking — `data_dir` works correctly.
